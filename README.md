# Fifth-Patient

> 一个以`hexo`框架搭建的个人博客

## hexo目录
```
.
├── _config.yml     // 网站的 配置 信息
├── package.json    // 应用程序的信息
├── scaffolds       // 模版文件夹
├── source          // 站点资源
|   ├── _drafts     // 草稿箱
|   └── _posts      // 博文
└── themes          // 主题文文件夹
```

## 功能

- [ ] RSS
- [ ] 评论
- [ ] 内容分享
- [ ] 搜索服务
- [ ] 查看图片
